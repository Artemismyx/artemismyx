周深介绍

周深（Charlie），1992年9月29日出生于湖南省邵阳市 ，成长于贵州省贵阳市，中国内地男歌手，毕业于乌克兰利沃夫国立立谢科音乐科学院美声专业 。 2014年，参加浙江卫视歌唱选秀节目《中国好声音第三季》，从而正式出道。2015年，发行首支个人单曲《玫瑰与小鹿》。2016年，获得东方风云榜最佳新人奖；同年，为动画电影《大鱼海棠》演唱印象曲《大鱼》，并凭借该单曲获得多个音乐奖项 [6-7]   [47-48]  。2017年，推出首张个人音乐专辑《深的深》。2018年，发行首支个人原创单曲《可它爱着这个世界》。2019年，参加音乐竞演节目《中国梦之声·我们的歌》，最终获得冠军 ；同年，开启“C-929星球”全国巡回演唱会。 2020年，作为首发歌手加盟《歌手·当打之年》，其演唱的《达拉崩吧》以破亿播放量斩获云村播放量第一；2020年起蝉联两届TMEA腾讯音乐娱乐盛典年度最佳内地男歌手奖。2021年，首次登上央视春晚，与张也合唱《灯火里的中国》；同年，为联合国《生物多样性公约》缔约方大会第十五次会议（COP15）演唱主题曲《和光同春》。2022年，加盟户外竞技真人秀《奔跑吧》担任常驻MC  ；与最高人民检察院合作推出的单曲《我的答案》获中宣部第十六届精神文明建设“五个一工程”奖。2023年，携个人单曲《花开忘忧》再度登上央视春晚舞台 。
