竹林中

    全景、侧面、跟摇，定场镜头（交代地点）

    此处一开始只闻其声，不见其人，给人神秘感让人沿着声音寻找其中的人 绵绵不断的雨声的音效，在竹林中箫声瑟瑟缠绕，将观众引入一个平淡清新的氛围之中。舒缓了紧张的情绪，也同时为电影之后大战到来铺垫了情绪。

    中景、正面

    镜头切到了正面，告诉我们有一个人单独来到了这里，好像要秘密做什么事情或者见什么人，大概率是见一个人，因为从这个画面开始就一直有一个箫声在指引他往哪走。这可能是整个电影的一个故事发生的推进或者转折。但这个时候我们还不知道他是谁，因为他低着头，一身比较隐蔽的打扮 （放） 这个时候他抬头了，我们知道了这个人是田战

    全景、侧面

    镜头转换到全景和侧面，在竹林中，影子正好坐在了两个最粗壮的竹子中间，也就是这个画面中的镜头重心的地方。这个时候我们知道了田战来这里是要见某一个人，一直在吹箫的那个人。

    全景、侧面、三分构图法

    镜头拉近了，我们知道了吹箫等待他的人是靖州，或者说在这时候田战的意识中，这是他的都督。 这个时候画面右边的竹子仍旧是最粗壮的，所以整个画面的视觉重心也是在靠右的影子身上。

    近景，侧面，影子

    吹箫的人抬头，是影子。影子是坐在画面中的右侧。

    近景，侧面，田战

    田战站在画面偏左的地方

    近景，侧面，给说话的人特写

    全景、侧面、跟摇

    镜头跟着影子起身的动作移动，影子走到了田战面前。并且这个时候，影子的身位也很好地卡在了两根竹子之间。

    近景，侧面，田战的特写，转头，在画面靠左的位置

    密室

    定场镜头（交代地点），跟摇

    画面一转来到了室内，人没有出现在镜头上，影子先出现在了墙壁上，镜头一直跟摇，画面中出现了人物

    中景，给了一个特写让影子打开密室，然后是田战站在影子身后好奇左顾右盼的镜头，交代了人物的位置关系，随后影子开门，让田战一个人进入。

    镜头给到大门的正面全景，这是一个地点的转换，这时田战推开大门，交代了故事关系，田战进入了密室，开始后面的情节。

    全景，镜头大部分颜色是灰暗的，因为是密室之内，都督背对着田战，一束光从上面的缺口打下来照到他身上，一动不动的像尊雕像，只有风吹动他的头发，显得他特别的孤独。

    摘下斗篷，显得很惊讶。

    近景，都督转过身

    镜头不动，人物慢慢转身，光线不变，显得他的脸很阴霾

    中景，正面，跟摇到侧面

    田战脱下披风跑到都督面前，镜头跟着他移动，当他跑到都督面前时，灯光依旧，灯光依旧打在都督身上，显得他瘦弱苍凉。

    关系镜头和过肩镜头

    都督拉住田战讲往事，来回都是关系与过肩镜头的切换。一个在左侧，一个在右侧。

    全景，侧面，二人面对面坐下。

    这个时候，这个时候唯一的光亮就是桌上的那个蜡烛，蜡烛和被它光照亮的两个人的脸颊，也是画面的视觉重心。

    关系镜头和过肩镜头

    仍旧是关系与过肩镜头的切换。都督左侧，田战右侧。 密室中是黑暗的，大部分光线都是顶光，有风吹动着都督的头发，体现了都督的野心、狡诈和丧心病狂。这样让角色更立体，有层次感，营造一种阴森与紧张的环境气氛。同时，更为对比密室外的世界提供了铺垫与反衬的作用。

    密室外

    田战走出密室（侧面中景，交代地点的转换）

    这个时候可以注意到，就是在田战这边的墙上和屏风上，都有境州的影子，他一直等在这里，这也告诉了我们此刻境州所站的方位。

    ！！！近景，大正面，影子特写

    这个镜头是整部电影中我最喜欢的镜头，也是因为这个镜头我才选取了这一段的前后情节完整的片段。

    此刻的靖州，也就是影子，如同一滩水墨画一样，渲染在屏风上，这里也是影片中对于屏风一处极好的应用，光从屋子的缝隙漏下来，人影映于屏风之上，透露着神秘的气息，也使人有了孰是孰非之感，一种诡谲的美感。因为我们都知道，影子是虚的，如同水墨画的晕染，真身是实的，如同水墨画的勾勒，这个时候晕染和勾勒模糊了彼此的边界，身体和影子融合在一起，虚虚实实，真真假假，难以估摸。 我在看完电影的时候一直在想，就是靖州是什么时候决定反客为主的，之前我以为是看到自己母亲被杀死的时候，但是后面第三第四遍看的时候，我觉得应该是在这里，伴随着画面的音效也在强调着这一刻。 他被头上的斗笠遮住了上半张脸，隐忍的站在屏风之后，看不清他的眼神，露出的下半张脸很阴郁，有些凶狠，让人感觉看不透他了，一种捉摸不透的危险，不像之前寄人篱下的影子，反而像那个阴鸷zhi4险恶的都督子虞。 也许这个时候，他就已经有了自己的想法，不再是那个一直为真身而活的影子，他有了自己的思考，自己想要的东西，以及自己想毁灭的东西。 应该是轮廓光，在画面中突出影子。打光是逆光的，等于说影子是面对黑暗，背照光明，这也许也能表达影子向死而生的心理。 而且结合前后镜头，这个镜头的视角应该是田战的第一视角，是田战看到的影子靖州，这也为之后他对影子靖州的态度、靖州对他说的话和最后影子杀了真身反客为主做下了铺垫。

    近景，侧面，田战看着影子

    这一刻田战的眼神一开始是有一种一半茫然一半不可置信的，然后换上了审视和考究的眼神去看影子。想象一下你刚刚知道有两个长得一样的人，你见的一直是其中一个的替身，你还没发现，知道了之后感觉这个替身比都督更像都督，真的会有一种，我的天呐的感觉。

    近景特写，大正面，影子抬头

    这里也是切回了田战的第一视角，这个镜头很简单，就是影子微微抬头，但是伴随着琴声，有一种扑面而来的压迫感。并且通过这个动作，我们也知道影子看到了田战出来。

    近景，侧面，跟摇，田战走出来

    交代田战方位的变换

    田战走到影子面前（中景）

    这个时候，通过镜头制造了一个帷幕，这应该是一个屏风的边框什么的东西，把画面一分为二。从他们的神情和姿态可以看出，他们之间还是有隔阂的，就是通过镜头的框架造就了一种隔阂感。田战站在屏风后面，用了一个光影的手法，把靖州的在屏风上的影子打在田战身上，让我们可以看到田战的神情。

    田战对影子说话

    田战难以置信地摇了摇头，然后发出了自己的感叹，这个时候田战对影子的态度，只是一个对待都督忠心下属的态度。在感叹的同时，也表达了对影子和子虞一模一样的肯定，甚至觉得影子比都督更像都督。

    影子的正面，田战的第一视角，一个内反打镜头

    这个时候二人之前还有屏风，也就是还有一层雾在，还有隔阂在，光影打在屏风上，也是侧面表达靖州的影子身份。这个时候的影子，我们是可以清晰地看到他的整张脸的，眼神晦涩。也许这个时候，露出整张脸的他，是那个仍旧忠心的影子。但是很快伴随着诡谲的音效，他低下了头，上半张脸重新陷入阴影，也就代表他回到了那个刚刚站在屏风后阴霾想反客为主的影子。嘴角似笑非笑说了一句模凌两可又引人深思的话。这句台词也确定了之后影子的命运和地位。

    一个过肩镜头，给了田战表情的一个特写

    表达出人物的距离感，一步一步拉近二人的位置，这时田战已经走过了屏风到达了影子面前。这个时候，二人之间不再有屏风，也不再有隔阂。田战对影子的点头，也表示他对刚刚影子说的那句话的肯定态度，为之后的事情做下铺垫。
