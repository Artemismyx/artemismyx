我最近在看汉尼拔

汉尼拔是美国作家托马斯·哈里斯的系列小说 《汉尼拔》系列里的主人公，是一位有着极高智力和敏锐洞察力的精神病医生，同时也是一个系列杀人犯，其出名之处在于他会吃掉被害人的一部分，所以也被称为"食人魔"。 汉尼拔·莱克特出身在立陶宛的一个贵族家庭，父亲是立陶宛的一位伯爵，叔叔罗伯特·莱克特是一个画家，汉尼拔还有一位妹妹，米莎。小时候的莱克特生活的无忧无虑，在莱克特城堡里过着传统贵族的生活，犹太裔的雅克夫成为了莱克特的启蒙老师，教会了莱克特许多几何知识和天文学知识，一切安静而又美好，但这一切在汉尼拔12岁时彻底改变，1944年-1945年的冬天，一架轰炸机轰炸了莱克特一家为了躲避战乱而临时居住的房屋，直接炸死了汉尼拔的父母和老师雅克夫，纳粹余党格鲁塔斯和其手下掳走了汉尼拔一家的财宝及幸存者汉尼拔和米莎，在接下来的那个难熬的冬天里，汉尼拔遇见了改变他一生的事情-----为了保命，对抗饥饿，格鲁塔斯和其手下砍死了米莎，并将其灭绝人性的分而食之，目睹了一切的汉尼拔精神受到了重创，变得沉默寡言。 经过了短暂的孤儿院生活，汉尼拔被已经继承了哥哥爵位的叔叔罗伯特接走，认识了自己的婶婶--紫夫人，一位高雅的日本女性，在此期间，汉尼拔冰封的心扉稍稍有些融化，可一切美好再次被夺走，市镇上的屠夫保罗出言侮辱紫夫人，瘦弱的叔叔想为妻子找回公道，却被强壮的屠夫推到了墙上，被钉子刺穿，导致了死亡。在受到了如此大的打击后，汉尼拔用一把剑完成了他的第一次杀人，划开了屠夫的肚子，割下了屠夫的脸颊肉就着菌菇吃了，并将屠夫的头放到了紫夫人的祭坛上，此后的5年间，汉尼拔在噩梦中成长，无时无刻不想着杀死残害自己妹妹的仇人，并在此后的时间里，逐一杀死了仇人，在此间他爱上了自己的婶婶，但即使是心爱的婶婶也无法阻挡汉尼拔复仇的步伐，在《汉尼拔崛起》的最后，汉尼拔杀死了残害妹妹的元凶格鲁塔斯，并得知了原来自己也吃了妹妹的肉才活下来，紫夫人也因为受不了汉尼拔的复仇行径而下定决心离开他。在书的末尾，汉尼拔迎着血色的夕阳，踏上了去美国的旅程。自此，汉尼拔也完成了从一位单纯少年向一个杀人不眨眼的恶魔的过渡。
