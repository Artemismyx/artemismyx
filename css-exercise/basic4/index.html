九尾介绍

2017年夏，许鑫蓁加入筹备中的Hero久竞战队；同年，许鑫蓁加入XQ俱乐部旗下青训战队——QST战队。2019年2月2日，许鑫蓁以上路选手的位置入选XQ战队2019KPL春季赛大名单； 3月9日，RNG.M对阵XQ，许鑫蓁转到中路，首登KPL赛场；XQ战队在常规赛阶段以4胜10负的成绩排名西部第7，无缘季后赛以及8月的世界冠军杯；9月至12月KPL秋季赛，常规赛XQ以10胜4负的成绩排名东部第3，11月16日XQ以0:4的比分落败RW侠，止步2019KPL季后赛六强；12月冬季冠军杯选拔赛，XQ战队积分不足，无法进入淘汰赛。2020年3月至6月KPL春季赛，XQ更名为广州TTG.XQ，常规赛阶段广州TTG.XQ以7胜8负的成绩排名东部第3，季后赛阶段广州TTG.XQ落败TS战队，止步春季赛四强；7月世界冠军杯，小组赛阶段广州TTG.XQ以3胜2负的成绩排名第3，同月31日广州TTG.XQ落败TS，止步世界冠军杯八强；9月KPL秋季赛，广州TTG.XQ更名为广州TTG，常规赛阶段广州TTG以8胜7负的成绩排名第8，季后赛阶段11月29日以0:4的比分落败WB.TS，止步季后赛六强。2021年1月14日王者荣耀冬季冠军杯，广州TTG以1:4的比分落败成都AG超玩会，止步2020王者荣耀冬季冠军杯八强；3月至6月KPL春季赛，许鑫蓁所在的广州TTG在常规赛获得S组第3，6月26日总决赛上，广州TTG以3:4的比分落败南京Hero久竞，获得2021KPL春季赛亚军；8月21日广州TTG以3:4的比分落败佛山GK，止步2021王者荣耀世界冠军杯四强；9月至12月KPL秋季赛，广州TTG在常规赛获得S组第5，在12月25日总决赛上，广州TTG以1:4的比分落败武汉eStarPro，获得2021KPL秋季赛亚军。2022年1月15日，广州TTG以1:4的比分落败武汉eStarPro，获得2021王者荣耀挑战者杯亚军； 2月KPL春季赛，常规赛广州TTG排名S组第五，进入季后赛败者组；4月23日，广州TTG以1:4的比分落败北京WB，许鑫蓁随队获得2022年KPL春季赛季后赛六强； 6月KPL夏季赛，常规赛广州TTG以2胜3负的成绩排名S组第五，进入季后赛败者组；8月21日，广州TTG以2:4的比分落败上海EDG.M，获得2022年KPL夏季赛季后赛六强。2023年1月，许鑫蓁入选TTG2023KPL春季赛大名单。5月10日，入选杭州亚运会电子竞技项目王者荣耀国家集训队候选人名单。
